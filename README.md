# im-qinyu
后端基于thinkphp6+swoole进行开发的即时通讯项目，前端使用uniapp进行开发（整体仿qq）

IM-ADMIN 为后台前端文件
IM-PHP 为项目api后台api文件
IM-UNIAPP 为项目前台Uniapp项目
